# MSQT - Backrun


## About

Bot that searches for arbitrage opportunities across unilike pools with backrunning trades

## Setup:

#### Install dependencies
```
npm install
```

#### Env vars
Create a `.env` file with params specified in `.env.sample`.

## Run:

#### Run forever
```
npm msqt-forever
```

#### Simple run
```
npm msqt
```

#### Run tests
```
npm test
```
